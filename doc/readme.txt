安装与配置：

1.双击打开UowMVC.sln，将项目“UowMVC.Web”设置为启动项目；

2.打开SQLSevrer（最低支持2005版本）,创建数据库；

2.下载完源代码后修改数据库连接字符串：“UowMVC.Web/Web.config 第13行

3.请同步更新“log4netconfig.xml”数据库连接字符串：“UowMVC.Web/log4netconfig.xml 第26行

  **重要：发布到IIS后请将“log4netconfig.xml”文件放到“Bin”目录下才能正常写入日志。

4.F5运行项目，输入账号密码，默认管理员密码：admin 123456，如果密码不正确，请打开“UowMVC.Repository/Migrations/Configuration.cs”文件查看初始化账号密码
